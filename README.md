# IDEA-Setting
IDEA 的基本设置

- 字体
IDE 用默认 Darcula 配置即可
编辑器使用 Fira Code Medium 字体
- 代码风格
缩进全部设置成 2（你喜欢 4 也行）
- 将 oneline 都勾选上
- 安装插件 Material Theme UI

## 快捷键

1. shift+shift 快速查询
2. ctrl+shift+g 全局检索
3. ctrl+f 当前文件检索
4. ctrl+shift+r 全局替换
5. ctrl+r 当前页替换
6. alt+` 版本控制
7. shift+f6 重命名文件/文件夹
8. alt+1234 文件、terminal、git working area、git history
9. ctrl+右击 查看代码引用
10. ctrl+\[ 回退代码引用
11. ctrl+\] 前进代码引用
